Franzi Ime Photography

VIEW WEBSITE

Franzi Ime Photography is a template design where I had to create a one page photography website for a photographer to display their images.

For this project, I want to display the photographer's work using a justified gallery inspired by Flickr. I implemented the feature thanks to the help of Miro Manniro's Justified Gallery plugin. I developed a feature to 'hide' certain photos depending on what the user wants to view.

What I Use

HTML5, CSS (SASS), Javascript, JQuery
