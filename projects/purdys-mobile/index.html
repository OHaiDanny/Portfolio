Overview

In 2013, Purdys Chocolatier began to bring their business into the world of mobile technologies. Purdys.com has to transition towards a mobile first approach to their existing website

By using the existing Purdys.com platform, A small team of developers and myself began to customize the website to accomodiate mobile technologies.

What I Use

HTML5, CSS3, JAVASCRIPT + JQUERY, ASP.NET, C#

My Duties

  • Collaborated with creative, marketing, and web sales departments to develop a responsive mobile design.
  • Developed .js files with developers to transform existing website for mobile compatibility.
  • Introduced various mobile design strategies and debugging techniques to optimize the usability of the website on current world devices.
  • Conducted group tests to accomodate website on all modern phone browsers.

My Impact

After rolling out the mobile version, it was met with positive feedback in terms of user friendliness and speed. We are continuing to improve Purdys Mobile to provide fast, alternate method for customers to buy what they want, where they want, when they want.

As for my experience, it was a fun challenge working with the constraints of Visual Studio to design a mobile website using current responsive practices and jQuery.
