The Mountain Top

VIEW WEBSITE

The Mountain Top is a template website that is focused on the idea of a restaurant business. This template uses a variety of javascript behaviours inspired by modern javascript focused websites.

I drew inspiration of using a parallax effect for the backgrounds of the website. I also implemented a sliding menu feature for the mobile version of the website.

What I Use

HTML5, CSS (SASS), Javascript, JQuery
