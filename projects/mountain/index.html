The Mountain Top

VIEW WEBSITE

The Mountain Top is a template website for a fictional restaurant. I use this template to play around with more advanced web design features.

I drew inspiration of using a parallax effect for the backgrounds of the website. I also implemented a sliding menu feature for the mobile version of the website.

What I Use

HTML5, CSS (SASS), Javascript, JQuery
