Ignite Designs

VIEW WEBSITE

Ignite Designs is a template project where my goal is to develop a simple one page website inspired by many websites that use a one column grid layout.

Some features that I developed is a retractable navigation menu for both mobile and desktop versions. I used a full height for each container of the page with jQuery scroll feature when a user clicks on a navigation item.

What I Use

HTML5, CSS3 (SASS), JavaScript, JQuery
